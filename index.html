Cat Puzzle

This game requires canvas to work properly. Your browser seems to do not support it. Please consider upgrading your browser if you want to play this game.

The game is quite simple : reconstruct the original image by swapping the puzzle tiles.

To move the tiles, simply drag-and-drop them around. The tiles will be swapped with each other.

To replay the game, just reload the page.

This game has been optimised for browser usage on a personnal computer.
